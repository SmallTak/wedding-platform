# 阶段一账号接口

## 登录与当前账号

- `POST /api/auth/login`：手机号密码登录。
- `GET /api/auth/me`：获取当前账号、角色、权限和首次登录状态。
- `PUT /api/account/password`：修改当前账号密码。
- `POST /api/account/avatar`：上传当前账号头像，支持 JPG、PNG 和 WebP，最大 5 MB。
- `PUT /api/account/profile`：完善姓名、头像、职位、服务地区和简介。

除登录外，请求需要携带 `Authorization: Bearer <token>`。

## 创作者账号管理

以下接口要求权限资源 `/accounts/creators`：

- `GET /api/admin/creators`：查询创作者账号。
- `POST /api/admin/creators`：开通创作者账号并分配一个或多个职业角色。
- `PATCH /api/admin/creators/{id}/status`：启用或停用账号。
- `POST /api/admin/creators/{id}/reset-password`：重置初始密码。
- `GET /api/admin/professional-roles`：查询可分配职业角色。

## 首次登录规则

创作者初次登录时 `setupRequired=true`，必须同时满足以下条件后才能进入工作台：

1. 修改管理员设置的初始密码。
2. 上传头像。
3. 填写姓名和职位。

前端路由守卫负责引导，后端分别保存 `must_change_password` 和 `profile_completed` 状态。
