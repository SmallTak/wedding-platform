婚礼项目功能已在 2026-07-20 随 Flyway `V15__remove_wedding_projects.sql` 从生产环境和当前代码中移除。

- 后端 `/api/projects`、`/api/admin/projects/*`、公开 `/api/public/projects*`、项目审核和客户项目关联接口均已下线。
- 工作台 `/console/projects` 导航和页面已删除。
- 数据库 `wedding_project`、`project_creator`、`project_customer_application` 表以及 `work_collection.project_id` 字段已由 V15 删除。

- 作品集已独立保存 `event_date`、`region_code`、`location_text`；后续内容维护请使用 `docs/阶段二作品集接口.md`。
