# 阶段三审核发布与官网接口

## 数据库版本

Flyway `V6__add_collection_review_metadata.sql` 为 `work_collection` 新增：

- `rejection_reason`：作品集资料当前驳回原因。
- `submitted_at`：最近提交审核时间。
- `reviewed_at`、`reviewed_by`：最近审核时间和管理员。
- 审核队列索引 `review_status + submitted_at`。

本阶段已于 2026-07-17 部署到生产环境，生产 schema 当前为 `V6`。官网、工作台、
公开分类、公开作品列表和审核中心路由已完成正式域名验收；未认证审核接口返回 `401`，
原图路径未公开并返回 `404`。

图片继续使用 `collection_photo` 现有审核字段。本阶段增加图片状态 `REJECTED`。

## 状态与并发规则

- 新作品集和新图片为 `DRAFT`。
- 创作者提交后，待审核图片变为 `PENDING`，作品集变为 `PENDING`。
- 管理员可批量通过或驳回待审核图片，驳回必须填写原因。
- 存在待审核图片时作品集保持 `PENDING`；存在驳回图片时为 `PARTIALLY_REJECTED`。
- 封面存在且全部未删除图片通过后，作品集进入 `APPROVED + READY`。
- 删除全部驳回图片后会重新计算状态；剩余图片全部通过且封面有效时自动进入 `READY`。
- 已通过图片不会因新图片重新提交而重复审核。
- 待审核期间修改作品资料、上传、排序、设置封面或删除图片，会撤回未处理图片并回到 `DRAFT`。
- 所有修改、提交、审核、发布和下架操作继续校验作品集 `version`。
- `PUBLISHED` 状态锁定创作者编辑；管理员下架后恢复编辑。

## 创作者提交

以下接口要求权限资源 `/content/collections`，创作者还必须是作品集创建人或参与者。

### `POST /api/collections/{collectionId}/submit`

请求：

```json
{
  "version": 3
}
```

提交前必须至少存在一张未删除图片，并设置有效封面。

响应包含：

- `collection`：最新作品集详情和状态。
- `photoBatch`：最新作品集版本、封面和全部未删除图片审核状态。

## 管理员审核与发布

以下接口要求管理员角色和权限资源 `/review/collections`。

### `GET /api/admin/reviews/collections`

分页查询审核与发布工作列表。支持：

- `page`、`size`
- `keyword`
- `reviewStatus`
- `publishStatus`

### `GET /api/admin/reviews/collections/{collectionId}`

查询作品集资料、创作者、图片和当前审核发布状态。

### `PUT /api/admin/reviews/collections/{collectionId}/photos`

批量审核当前作品集中的待审核图片。

```json
{
  "version": 4,
  "photoIds": [11, 12],
  "decision": "APPROVE",
  "reason": null
}
```

`decision=REJECT` 时 `reason` 必填。

### `POST /api/admin/reviews/collections/{collectionId}/approve`

当作品集没有待审核或驳回图片时，通过作品集资料并进入 `READY`。

### `POST /api/admin/reviews/collections/{collectionId}/reject`

驳回作品集资料，请求包含 `version` 和必填 `reason`。

### `POST /api/admin/reviews/collections/{collectionId}/publish`

```json
{
  "version": 6,
  "visibility": "PUBLIC",
  "featured": true,
  "pinned": false,
  "sortOrder": 10
}
```

只有 `APPROVED + READY`、全部图片已通过且封面有效的作品集可以发布。
当前支持 `PUBLIC` 和 `HIDDEN`，`PASSWORD` 会明确返回暂不支持。

### `POST /api/admin/reviews/collections/{collectionId}/offline`

请求包含 `version` 和必填 `reason`。下架后公开接口立即不可访问，创作者可以继续编辑。

### `GET /api/admin/dashboard/overview`

返回待审核、部分驳回、可发布、已发布数量和最近审核列表，用于工作台真实运营概览。

## 官网公开接口

以下接口无需登录，只返回 `PUBLISHED + PUBLIC` 内容以及审核通过的水印预览图。

- `GET /api/public/content/categories`
- `GET /api/public/collections`
- `GET /api/public/collections/{collectionId}`

公开作品列表支持 `page`、`size`、`keyword` 和 `categoryId`。排序规则为置顶优先、
手动顺序优先、发布时间倒序。

公开响应不包含原图路径、文件摘要、创建账号或内部审核信息。已下架、隐藏和未发布作品
统一按公开内容不存在处理。

## 页面入口

- 工作台审核中心：`/console/reviews`
- 创作者提交审核：`/console/collections/{collectionId}/photos`
- 官网作品首页：`/`
- 官网作品详情：`/collections/{collectionId}`
